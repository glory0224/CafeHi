# ☕ CafeHi PROJECT ☕

### 사용된 기술 스택

<li>프로그래밍 언어 : Java 11 , thymeleaf </li>

<li>데이터베이스 : Oracle 11g </li>

<li>프레임워크 : Spring Boot , BootStrap 5.2.0 </li>

<li> IDE : IntelliJ </li>


### 구현된 기능 사항

1. 회원가입, QnA CRUD 

2. 카카오 기반 위치 제공  

3. 정규표현식을 이용한 로그인 검증

4. 게시글 페이징 

5. 게시글 검색 

6. 아이디 중복 확인 

7. 스프링 시큐리티 

8. 회원 가입 

9. 마이 페이지  

10. 장바구니  

11. 계정 찾기 (아이디, 패스워드)

12. 주문 

13. 멤버쉽 적립
